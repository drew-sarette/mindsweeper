mindsweeper

how to play

Enter a number from 1 to 100 to start a new Mindsweeper game. Click/tap and hold to flag a square as a possible mine, and repeat to unflag and check the same square. To win, you must click on each square that does not contain a mine! In memory mode, the number of adjacent mines will only appear temporarily. Good luck!
